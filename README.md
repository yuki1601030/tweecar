# CarTweet Prototype

CarTweet は、車同士がクラウド上でリアルタイムに大量の情報をやり取りし、自車AIが運転者に必要な情報だけを要約して届ける「車専用SNS」のユーザーテスト用 Web プロトタイプです。

説明資料ではなく、PC上で「運転中の車載UIを触っている感覚」を確認するための体験型プロトタイプです。データはすべてモックで、実在の個人情報は含みません。

## 起動方法

```bash
npm install
npm run dev
```

ビルド確認:

```bash
npm run build
```

## プロトタイプの目的

- 車同士が裏側で会話し、人間にはAI要約だけが届く世界観を体験してもらう
- 説明を読ませるのではなく、カード判断・共有操作・ルート反映を通じて利用シーンを想像してもらう
- Privacy Guard により、生投稿や個人情報を見せない設計の安心感を確認する
- 安全支援だけでなく、休憩、景色、道路管理、観光などへの広がりを短く提示する

## ユーザーテストで確認したいこと

- この世界観は自然か
- 運転中に使うイメージが持てるか
- どんな情報が有益か
- どの操作がしやすいか
- 情報量は多すぎないか
- プライバシー面に安心感があるか

## GitHub Pagesで公開する場合の注意

- Vite の静的ビルドは `npm run build` で `dist/` に出力されます。
- リポジトリ名のサブパスで公開する場合は、必要に応じて `vite.config.js` で `base: '/リポジトリ名/'` を設定してください。
- 追加ライブラリは使っていません。公開前にモック文言・画像・データに実在の個人情報が含まれないことを確認してください。
