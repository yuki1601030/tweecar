# TweeCar プロトタイプ

TweeCar は、街中の車たちが裏側で匿名情報を共有し、自車AIがドライバーに必要な情報だけ届けるサービスのユーザーテスト用プロトタイプです。

## 画面構成

- **サービスイメージ画面**  
  街全体、道路を走る車、車たちの吹き出し、AIおすすめパネル、運転手への最終通知でサービスの世界観を短時間で伝える画面です。

- **ユーザー体験画面**  
  左に車内ディスプレイ、右にスマホチャット、下部にクイック質問ボタンを配置しています。説明を読ませるよりも、実際に車へ話しかけているような操作感を重視しています。

## 体験の前提

- ドライバーは他車の投稿を直接見ません。
- TweeCar AI が匿名情報を読み取り、必要な道路・駐車場・休憩情報だけを知らせます。
- 音声入力中心で利用し、話した内容やAIの回答はスマホチャットにも連動する想定です。
- このプロトタイプでは外部API、ログイン、DBは使わず、ダミーデータで動作します。

## GitHub Pagesでの確認方法

`index.html` 1ファイルで動作します。ビルドは不要です。

1. このリポジトリを GitHub に push します。
2. GitHub の **Settings → Pages** を開きます。
3. Source を対象ブランチ、Folder を `/ (root)` に設定します。
4. 発行された GitHub Pages URL を開くと、そのままプロトタイプを確認できます。

ローカルでは `index.html` をブラウザで直接開くか、任意の静的サーバーで確認できます。
