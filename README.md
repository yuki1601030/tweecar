# CarTweet Live

CarTweet Live は、車専用SNSの世界観をユーザーテストで視覚的に体験するための React + Vite 製Webプロトタイプです。車が走行中に見たこと、感じたこと、道路や店舗・景色の変化を、位置情報つきでリアルタイムにつぶやく体験を1画面で確認できます。

## 起動方法

```bash
npm install
npm run dev
```

ビルド確認:

```bash
npm run build
```

## このプロトタイプの目的

- 「Twitter/Xの車版」のように、車が匿名で大量につぶやく世界観を直感的に伝える
- 危険情報だけでなく、景色、店舗、SA/PA、混雑、道路状態、日常の違和感や雑談も投稿されるイメージを検証する
- 安全支援アプリに寄せすぎず、SNSらしいライブ感とプロダクト感を優先する
- 将来のビッグデータ活用につながる投稿データの広がりを説明しやすくする

## ユーザーテストで確認したいこと

- 車が自動でつぶやく世界観は自然か
- どんな投稿なら見たいか
- どんな投稿は不要か
- 写真や位置情報があると信頼できるか
- 自分も投稿したいと思うか
- データ活用に抵抗があるか
- 安全、観光、店舗、道路管理、保険、防災のどこに価値を感じるか

## 将来構想

CarTweetで集まる全国の車の投稿データを、CarTweet Data Cloudに蓄積し、AIで解析することで以下のようなサービス展開を想定しています。

- リアルタイム安全支援
- 道路管理/自治体向けレポート
- 観光スポット発見
- 店舗送客/広告
- 保険リスク分析
- 災害時の現地情報収集
- 車オーナー向け体験提案

## GitHub Pagesで公開する場合の注意

- Viteの静的ビルドは `npm run build` で `dist/` に出力されます。
- リポジトリ名のサブパスで公開する場合は、必要に応じて `vite.config.js` で `base: '/リポジトリ名/'` を設定してください。
- 追加ライブラリは使わず、表示データはすべてモックです。社内情報や実在の個人情報は含めないでください。
